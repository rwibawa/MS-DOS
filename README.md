## 1. Setup:
```sh
$ sudo apt install -y vulkan-tools
$ vulkaninfo 
$ vkcube
$ sudo apt install -y libvulkan-dev
$ sudo apt install -y vulkan-validationlayers-dev spirv-tools
```
